# LibraryManagementSystem #java #jdbc #mysql

Project Overview:
This Library Management System is a console-based application developed using Java, JDBC, and MySQL.
It allows librarians/admins to manage books, users, and borrowing records easily.

Features:

1.User Management:
 Add new users,
 View user details,
 Update or delete users
 
2.Book Management:
 Add new books,
 Search books by title/author,
 Update book details,
 Delete books
 
3.Borrow/Return System:
 Borrow a book,
 Return a book,
 View all borrowed books,
 Track due dates

Technologies Used:
 Java(Core Java, OOPS, Collections),
 JDBC(Java Database Connectivity),
 MySQL for database,
 IntelliJ IDEA as IDE
